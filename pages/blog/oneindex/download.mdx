# 共享网盘下载说明

总的来说，因为 OneDrive 对于单个直链设置了并发限制，所以使用 IDM 这种多线程下载工具可能会使同时下载同一个资源的其他群友带来困扰，他们可能会报错无法下载。理论上说直链下载的速度可以拉满带宽，但是实测你科校园网大概只能维持在 1MB/s~2MB/s（当然比百度网盘限速后快）。

但是鉴于该共享网盘只服务你群群友，所以大概也不会有什么问题；只是在使用下载工具前三思。

此外，OneDrive 的每一个直链都有一定的生命周期，虽然 IDM 等下载工具可能会标注支持断点续传，但是也务必要在一个链接失效之前完成下载。

## 使用同步客户端

虽然绝大多数群友会使用 Index 页或者是网页版 OneDrive 下载文件，但是考虑到仍有管理员使用本地同步客户端，故作此提醒：

**不要直接拖动 OneDrive 文件夹中的文件到你的硬盘中！这可能会被视为剪切而导致 OneDrive 中的文件丢失！**

### 修改 host 文件

如果出现 OneDrive 的域名被污染的情况，可能可以通过向 host 文件中增加下面的内容解决。

```
13.107.136.9 aoikaze-my.sharepoint.com
13.107.138.9 aoikaze-my.sharepoint.com
52.105.229.41 aoikaze-my.sharepoint.com
```

host 文件一般位于 `C:\Windows\System32\drivers\etc` 目录下，需要先复制出来，修改之后再覆盖回去。
