# 关于群共享网盘

使用 OneDrive Business for E5 Developer 作为存储空间；

使用 **[onedrive-vercel-index](https://github.com/spencerwooo/onedrive-vercel-index)** 生成站点，并部署至 [Vercel](https://vercel.com)；

## 与之相关的账号

这些账号一般由群主保管；如果由特殊需要可以去找群主索要

| 账号        | 用途                                                         |
| ----------- | ------------------------------------------------------------ |
| 微软账号    | 1）注册 Github 账号<br />2）白嫖 Office E5 管理员帐号        |
| Github 账号 | 1）白嫖 Vercel、Upstash、Github Pages<br />2）更新群公告导引 |
| E5 子账号   | 由各管理持有以分发共享链接                                   |
